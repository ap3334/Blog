## Blog Project
- A project to build a blog that supports simple functions

---

**1. 프로젝트 기간** : 2022.01.18 ~ 2022.01.30 (약 2주)

**2. 개발 환경** : IntelliJ

**3. 개발 언어 / 라이브러리** 

- 간단한 HTML, JS, JSP 활용
- MySQL 사용
- Java, Spring Framework를 활용

**4. 구현 기능**

- 회원가입 : AJAX 사용
- 로그인 : 스프링 시큐리티로 구현
- 게시글, 댓글 작성 / 삭제 구현
- 카카오 로그인 서비스
- 회원 정보 수정
